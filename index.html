PumpTrack is a workout tracking app designed for BodyPump® classes, BodyPump® Heavy sessions, and bodybuilding workouts. This guide walks you through all features.

New user? Start with Quick Start (5 minutes) for the fastest setup and first workout flow.

Getting Started

Use this section for your first-day setup, then move to session logging. The app auto-saves throughout normal use.

When you first open PumpTrack, you'll see the Home screen with:

  • Start New Session button to begin a workout
  • Recent Sessions list showing your latest workouts
  • Navigation links to:
    • About PumpTrack
    • Workout Calendar
    • Measurements
    • View Full History
    • Exercise catalog

Home Screen Overview

  • Tap Start New Session to create a new workout
  • Tap any recent session to continue or review it
  • Tap the gear icon (top right) to access Settings

Quick Start in 5 Minutes

Follow this short path if you want to log your first workout right away.

  1. Open Settings and set your default weight unit (LB or KG).
  2. In Gear Setup, add your available plates and dumbbells.
  3. Return Home and tap Start New Session.
  4. Select your workout type and tap Create Blank Session.
  5. Add at least one exercise with +, then enter weight and effort.
  6. Complete the workout and verify it appears in View Full History.

Tip: If you train a similar format weekly, use Clone workout to speed up session setup next time.

How the App Works

PumpTrack organizes workouts around the structure of the class or program you're doing — not a generic exercise list. Understanding this structure makes the rest of the app click into place.

Tracks and Sections

A BodyPump® class is divided into tracks — Warmup, Squats, Chest, Back, Triceps, Biceps, Lunges/Glutes, Shoulders, and Core. Each track targets a specific body part and typically uses a barbell, dumbbells, or both.

PumpTrack mirrors this exactly. When you create a BodyPump® session, the app builds one section per track. Inside each section you add the exercises you're doing — usually one per track, but you can add more. After each track in class, you open that section and log your weight and effort.

Exercises and Entries

An exercise is a movement from your library (e.g., "Barbell Squat" or "Dumbbell Curl"). An entry is a single logged instance of that exercise in a session — with a weight, unit, effort score, and optional notes.

When you clone a past session, all your exercises carry forward with last week's weights pre-filled. You log on top of that: match, increase, or note a change.

Session Types

  • BodyPump® — Nine tracks in fixed order, one section per track. Best for standard Les Mills classes.
  • BodyPump® Heavy — Same tracks, but grouped into blocks: clusters of tracks done in sequence before a full rest. You define the block structure before the session.
  • Bodybuilding — Sections by muscle group, no fixed order. Choose which groups to include for each session.
  • Custom — Free-form. Add any exercises to a single open session.

The Core Loop

  1. Clone last week's session (or create a new one).
  2. Check the pre-class gear list and set out your plates.
  3. During class, log weight and effort for each track as it ends.
  4. Tap Complete Workout when done.
  5. Next week, clone again — your weights are waiting.

Creating a New Session

  1. From Home, tap Start New Session
  2. Fill in session details:
    • Workout name — Edit the default name or enter your own
    • Workout type — Choose from:
      • BodyPump® — Standard BodyPump® class format
      • BodyPump® Heavy — Heavy format with block planning
      • Bodybuilding — Custom bodybuilding workout
      • Custom — General workout
    • Date — Select the workout date
    • Notes — Optional notes for the session
  3. Tap Create Blank Session to start

Bodybuilding Options

When selecting Bodybuilding, you can choose which muscle groups to include:

  • Tap Visible groups to select/deselect body parts
  • Only selected groups appear in your workout
  • You can change this later from session detail

BodyPump® Heavy Options

When selecting BodyPump® Heavy, you define blocks before the session starts. A block is a group of tracks done in sequence — for example, Chest + Back + Squats cycled through before a full rest.

The default plan has five blocks (Warmup + four working blocks). To customize:

  1. Tap Block combinations to open the block designer
  2. Tap any track name within a block to change it to a different track
  3. Tap + Add block to add a new block to the plan
  4. Tap the − button on a block to remove it
  5. Tap Reset to return to the default plan
  6. Tap Save when done

Tip: You can also redesign blocks from within an open session using the Redesign blocks option in session detail.

Logging Your Workout

Viewing Your Workout

When you open a session, you'll see:

  • Workout name and type at the top
  • Date (tap "Edit" to change)
  • Pre-class gear checklist (for BodyPump®/BodyPump® Heavy)
  • Exercises organized by track/body part
  • Session notes at the bottom

Exercise Sections

Each track/body part is a collapsible section:

  • Tap the chevron to expand/collapse a section
  • Tap + to add an exercise to that section
  • Tap the checkmark in the section header to complete all exercises in that section at once

Adding Weight and Effort

  1. Tap an exercise row to open the entry editor
  2. Enter:
    • Weight — Use the number pad or quick-weight chips
    • Effort (1–10) — Your RPE rating for this exercise (see below)
    • Notes — Optional notes for that exercise
  3. Tap the unit chip (LB/KG) beside the weight field to switch units
  4. Weight and effort are auto-saved as you enter them

Understanding Effort (RPE 1–10)

Effort is a Rate of Perceived Exertion (RPE) score — how hard the exercise felt on a 1–10 scale:

  • 1–4 — Easy, warm-up level; you could keep going indefinitely
  • 5–6 — Moderate, working but comfortable
  • 7–8 — Hard; the last few reps are challenging
  • 9–10 — Near-maximum or maximum effort

For BodyPump® working tracks, most people land between 6 and 9. Both weight and effort are required before the app will let you mark an exercise complete — effort alongside weight tells a more complete story than weight alone.

Quick Weight Chips

The entry editor shows quick-select chips for common weights:

  • Chips are based on your configured plate and dumbbell options in Settings
  • Available when editing barbell or dumbbell exercises
  • If chips are missing, check that your gear inventory is set up in Settings → Gear Setup

Marking Exercises Complete

  • Tap the circle icon next to an exercise to toggle completion
  • Requirement: weight AND effort (1–10) must both be entered
  • If you try to complete without both, you'll see an error message
  • Tap the checkmark in a section header to complete all exercises in that section at once

Previous Weight Recall

When you tap an exercise, PumpTrack automatically shows your previous weight for that exercise — "Last used: X lb." This is pulled from the most recent matching session.

Tip: Previous weights are matched by track name and exercise order. Use Clone workout rather than starting a new session to keep this matching reliable.

Working with Exercises

Adding a New Exercise

  1. Tap the + button in any section
  2. Choose how to add:
    • Search the library — Find an existing exercise
    • Custom — Enter a new exercise name
  3. The new exercise is automatically added to your library for future use

Changing an Exercise

  1. Tap an exercise row to open the editor
  2. Tap Change exercise...
  3. Select a different exercise from the library
  4. Your weight/effort is preserved unless you clear it

Reordering Exercises

  • Within a section: Long-press and drag to reorder

Removing an Exercise

  1. Tap the exercise row to open the editor
  2. Scroll down and tap Remove exercise
  3. Confirm when prompted

Pre-Class Gear Checklist

The gear checklist appears at the top of any BodyPump® or BodyPump® Heavy session. It calculates exactly which plates and dumbbells you'll need for the entire workout — so you can set everything out before class starts, rather than scrambling between tracks.

Setup (Do This Once)

The checklist needs to know what equipment you have before it can build the list. In Settings → Gear Setup:

  1. Set Barbell weight input to match how you enter weights:
    • Per side — You enter the weight loaded on each side (e.g., 20 kg means 20 kg per side)
    • Total barbell — You enter the full loaded weight including the bar
  2. Under Available plates, add every plate size at your gym
  3. Under Available dumbbells, add the dumbbell weights available to you

How the Math Works

For each exercise in your session that has a weight logged, the checklist works out which plate combination makes up that weight using the sizes you've configured. It then builds a consolidated list across the whole session.

Reuse is accounted for: if two sequential exercises use the same plate configuration, the plates are counted once — not duplicated. You carry what you actually need to set out, not a worst-case total.

When the Checklist Looks Wrong

  • No items appear: You may not have weights logged yet, or gear inventory isn't configured. Open Settings → Gear Setup and confirm plates and dumbbells are added.
  • Wrong plate breakdown: Check that the barbell input mode (per-side vs. total) matches how you're entering weights in the entry editor.
  • An exercise is missing: Only exercises with a weight entered appear in the checklist. Exercises with no weight logged are excluded.

Completing a Workout

Complete All Tracks

When you've logged weight and effort for all exercises:

  1. Scroll to the bottom of the session
  2. Tap Complete Workout
  3. All eligible exercises are marked as complete
  4. The session is saved to your history

Clone a Workout

Need to repeat a workout? Use Clone:

  1. Tap More... menu
  2. Select Clone workout
  3. A copy is created with all exercises and last session's weights, but completion cleared

Refresh from Latest

Updates the weights in the current session from your most recent matching workout, without creating a new session:

  1. Tap More... menu
  2. Select Refresh from latest
  3. Matching exercises are updated with previous weights and effort

Delete a Workout

  1. Tap More... menu
  2. Select Delete workout
  3. The workout is moved to "removed" status
  4. You can recall it within 30 days from History

Viewing History

Access View Full History from the Home screen:

History Features

  • Filter by workout type — Tap the filter to show All/BodyPump®/BodyPump® Heavy/Bodybuilding/Custom
  • Show removed — Toggle to see deleted workouts you can recall
  • Export CSV — Export your history for spreadsheet analysis
  • Edit mode — Swipe to delete or permanently remove workouts

Editing a Past Workout

History is read-only by default to prevent accidental edits. To modify a past session:

  1. Tap the workout in History
  2. Tap Edit in the top-right
  3. Make your changes — they save automatically

Tip: If you want to experiment without affecting the original, use Clone instead of Edit.

Recalling a Deleted Workout

  1. Enable Show removed in History
  2. Tap the deleted workout
  3. Tap Recall to restore it to your history

Body Measurements

Track your body measurements over time:

Adding Measurements

  1. From Home, tap Measurements
  2. Tap the + button
  3. Enter:
    • Date — Date of measurement
    • Unit — Inches or centimeters
    • Body weight unit — lb or kg
    • Measurements — Enter values for any body parts:
      • Body weight, Waist, Hips, Chest, Shoulders
      • Upper arm (relaxed/flexed), Forearm
      • Thigh, Calf, Neck
    • Notes — Optional notes
  4. Tap Save

Viewing Progress

  • Each entry shows change vs. previous measurement
  • Entries are sorted by date
  • Tap any entry to view details or edit

Exercise Catalog

The exercise catalog is your library of movements. Every exercise you've added to a session — whether picked from suggestions or entered as a custom name — lives here.

Accessing the Catalog

From Home, tap Exercise catalog

Features

  • View all exercises in your library
  • Tap an exercise to rename it
  • Renaming updates the exercise name across all past and future sessions that reference it
  • Exercise names must be unique (case-insensitive)

Suggested Exercise Lists

The app suggests exercises for each track when you tap + in a session, based on what you've used there before. To rebuild these suggestions after renaming exercises or changing your exercise habits:

  1. Go to Settings → Maintenance
  2. Tap Rebuild suggested exercise lists from workout history
  3. Confirm when prompted

Settings

Access Settings from the gear icon on the Home screen:

Default Weight Unit

Choose LB or KG as the default unit for new exercise entries. You can still override the unit per exercise in the entry editor.

Important — unit toggling does not convert numbers. Switching the unit on an existing entry retags the unit label only. If you have 45 LB and tap the unit chip, it becomes 45 KG — not 20.4 KG. Use this to correct a mislabeled entry. If you need to convert, switch the unit first, then update the number manually.

Workout Calendar

  • Show Workout Calendar on Home — Toggle visibility
  • Workout type colors — Customize colors for each workout type

Gear Setup

Configure your available equipment for the pre-class gear checklist:

  • Barbell weight input — Choose:
    • Per side — Enter weight per side (e.g., 20 kg per side = 40 kg total)
    • Total barbell — Enter the full loaded weight including the bar
  • Available plates — Add/remove plate sizes (kg or lb). Add every plate size at your gym.
  • Available dumbbells — Add/remove dumbbell weights. These also power the quick-weight chips in the entry editor.

Maintenance

  • Rebuild suggested exercise lists — Refresh exercise suggestions based on history

Exporting Data

Export your workout history as CSV for use in spreadsheets:

  1. Go to View Full History
  2. Tap Export CSV in the toolbar
  3. Choose how to share (AirDrop, email, save to Files, etc.)

The CSV includes all workout data organized by session, exercise, weight, effort scores, and notes.

Data Management and Safety

Before You Delete

  • Deleting a workout moves it to removed status first
  • Removed workouts can be recalled from History within 30 days
  • Permanent deletion cannot be undone

Recommended Backup Routine

PumpTrack stores all data locally on your device. Until iCloud sync is added, your data lives in the app only and is not automatically backed up. Export regularly:

  1. Export CSV from View Full History once per week (or after major training blocks)
  2. Save the file to cloud storage (iCloud Drive, Google Drive, Dropbox, etc.)
  3. Use a consistent filename like PumpTrack_YYYY-MM-DD.csv

Data Quality Tips

  • Keep exercise names consistent to improve suggested exercise lists
  • Avoid duplicate custom names with different capitalization
  • Run Rebuild suggested exercise lists after many exercise renames

Troubleshooting

I can't mark an exercise complete

Both weight and effort (1–10) are required. Confirm both are entered. Effort must be a number from 1 to 10.

Quick-weight chips are missing

Chips appear only for barbell and dumbbell exercises. Check that plates and/or dumbbells are configured in Settings → Gear Setup.

Previous weights aren't showing

Previous weights are matched by track name and exercise order within that track. If you renamed an exercise, changed the order of exercises, or created a fresh session instead of cloning, the match won't be found. Use Clone workout to preserve matching reliably.

The gear checklist is empty or looks wrong

See the Pre-Class Gear Checklist section. The most common causes: no gear inventory configured, wrong barbell input mode, or no weights logged yet.

A deleted workout is not visible

Go to View Full History and enable Show removed. If more than 30 days have passed, the item may no longer be recallable.

The unit on an entry looks wrong

Tap the exercise row and use the unit chip in the entry editor to retag it. Note: this changes the label only, not the number. Correct the number manually if needed.

Suggested exercises seem outdated

Go to Settings → Maintenance and run Rebuild suggested exercise lists from workout history.

FAQ

How do I set up for my first BodyPump® class?

Go to Settings → Gear Setup first: set your barbell input mode and add your plate and dumbbell sizes. Then create a new BodyPump® session. After your first class, clone that session each week — your exercise order and previous weights carry forward automatically.

Why aren't my previous weights showing?

Weights are matched by track name and exercise position within that track. If you created a fresh session instead of cloning, changed the order of exercises, or renamed an exercise, the match breaks. Always use Clone workout to keep history matching reliable.

Why can't I mark an exercise complete?

Both weight and effort (1–10) are required. Enter both values and the completion circle becomes tappable.

I switched the unit from LB to KG and the number looks wrong. What happened?

Switching the unit retags the entry — it doesn't convert the number. 45 LB becomes 45 KG, not 20.4 KG. Tap the unit chip again to switch back, or update the number manually to the correct value.

The gear checklist looks wrong. What should I check?

First, confirm your plate and dumbbell inventory is set in Settings → Gear Setup. Second, check that your barbell input mode (per side vs. total) matches how you enter weights. Exercises with no weight logged don't appear in the checklist.

I deleted a workout by accident. Can I recover it?

Yes, within 30 days. Go to View Full History, enable Show removed, tap the workout, and tap Recall.

Does PumpTrack auto-save?

Yes. Weight, effort, notes, unit changes, and completion state all save automatically as you enter them. You can leave mid-session and everything will be there when you return.

Can I use PumpTrack for non-BodyPump® workouts?

Yes. Use Bodybuilding for muscle-group-based lifting or Custom for anything else. Both support multiple exercises per section and the same weight/effort logging.

What's the fastest way to repeat a workout?

Open the previous workout from Home or History, tap More..., and choose Clone workout. The clone opens with all your exercises and last week's weights pre-filled.

Can I export my data?

Yes. Use Export CSV from the full History screen. The file includes all sessions, exercises, weights, effort scores, and notes.

Glossary

Track
A named portion of a BodyPump®-style session targeting one body part — for example, Squats, Chest, or Core. Each track is one section in the app.
Block
A group of tracks done in sequence without full rest between them, used in BodyPump® Heavy sessions. For example, a block might pair Chest + Back + Squats done three times through before a rest.
Entry
A single logged instance of an exercise in a session — includes weight, unit, effort score, and optional notes.
Effort (RPE 1–10)
Rate of Perceived Exertion: your subjective intensity rating, where 1 is barely any effort and 10 is maximum. Both weight and effort are required to mark an exercise complete.
Clone Workout
Creates a copy of an existing workout with the same exercise structure and previous weights, but with completion cleared. The recommended way to repeat a weekly session.
Refresh from Latest
Updates matching exercises in the current session with your most recent logged values from a similar past workout.
Removed Workout
A soft-deleted workout that can be recalled from History within 30 days before permanent deletion.
Exercise Catalog
Your library of exercise names, used for search, reuse, and consistency across sessions.
Per Side
Barbell input mode where you enter the weight loaded on each side of the bar. The gear checklist uses this to calculate the correct plate combination.

Class-Day Quick Reference

Use this checklist on training day. You can print just this page using the button below.

Before Class (5 minutes)

  • Open today's session or clone last week's from Home.
  • Expand the Pre-class gear checklist at the top of the session.
  • Set out your plates and dumbbells based on the checklist.
  • Glance at last week's weights to know what you're aiming for.

During Class (after each track)

  • When a track ends, tap that section to expand it.
  • Enter weight and effort (1–10) while it's fresh — don't wait until the end of class.
  • Tap the section checkmark to complete the whole track at once.

After Class (2 minutes)

  • Add session notes — what felt strong, what to adjust, any weights to change next time.
  • Tap Complete Workout to lock in the session.
  • Verify it appears in View Full History.

Tips and Best Practices

For BodyPump® Classes

  • Clone, don't create. Use Clone workout each week instead of starting fresh — your exercise order and previous weights carry forward, and previous-weight recall stays reliable.
  • Log each track when it ends, not at the end of class. Memory degrades under fatigue; a quick entry right after each track is far more accurate.
  • Use the section checkmark (tap the track header) to complete all exercises in a track at once — faster than tapping each row individually.
  • Set up your plate inventory once in Settings so the gear checklist works automatically every week.

For BodyPump® Heavy

  • Design your blocks before class, not during warm-up. Open the block designer when creating the session.
  • The gear checklist is especially useful for Heavy — you'll often need more plate combinations than a standard class.
  • You can redesign blocks from within an open session if the structure changes on the day.

For Bodybuilding

  • Select only the muscle groups you're working that day to keep the session view clean.
  • Add multiple exercises per body part using the + button in each section.
  • Use session notes to record supersets, rest times, or anything that doesn't fit in the weight/effort fields.

General Tips

  • The app auto-saves as you type — no manual save needed.
  • Export CSV regularly and save to cloud storage. Until iCloud sync is added, this is your only backup.
  • Use the calendar on Home to see your workout frequency at a glance.
  • Effort scores over time tell you whether a weight is getting easier — a clear sign to increase next week.
