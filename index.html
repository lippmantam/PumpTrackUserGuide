PumpTrack is a workout tracking app designed for BodyPump® classes, BodyPump® Heavy sessions, and bodybuilding workouts. This guide walks you through all features.

New user? Start with Quick Start (5 minutes) for the fastest setup and first workout flow.

Getting Started

Your first-day setup and a tour of the Home screen.

Use this section for your first-day setup, then move to session logging. The app auto-saves throughout normal use.

When you first open PumpTrack, you'll see the Home screen with:

  • Start New Session button to begin a workout
  • Recent Sessions list showing your latest workouts
  • Navigation links to About PumpTrack, Workout Calendar, Measurements, View Full History, and Exercise catalog

Home Screen Overview

  • Tap Start New Session to create a new workout
  • Tap any recent session to continue or review it
  • Tap the gear icon (top right) to access Settings

Quick Start in 5 Minutes

From zero to your first logged workout in five minutes.

  1. Open Settings (gear icon on Home) and set your default weight unit (LB or KG).
  2. In Gear Setup, add your available plates and dumbbells.
  3. Return Home and tap Start New Session.
  4. Select your workout type and tap Create Blank Session.
  5. Add at least one exercise with +, then enter weight and effort.
  6. Complete the workout and verify it appears in View Full History.

If you train a similar format weekly, use Clone workout to speed up session setup next time.

How the App Works

The structure behind sessions, tracks, and the weekly training loop.

PumpTrack organizes workouts around the structure of the class or program you're doing — not a generic exercise list. Understanding this makes the rest of the app click into place.

Tracks and Sections

A BodyPump® class is divided into tracks, each targeting a specific body part. PumpTrack mirrors this exactly: one section per track, with exercises you add inside each one. After each track in class, you open that section and log your weight and effort.

Exercises and Entries

An exercise is a movement from your library (e.g., "Barbell Squat"). An entry is a single logged instance in a session — weight, unit, effort score, and optional notes. When you clone a past session, all entries carry forward with last week's weights pre-filled.

Session Types

  • BodyPump® — Nine tracks in fixed order. Best for standard Les Mills classes.
  • BodyPump® Heavy — Same tracks grouped into blocks done in sequence before a full rest. You define the block structure before the session.
  • Bodybuilding — Sections by muscle group, no fixed order. Choose which groups to include each session.
  • Custom — Free-form. Add any exercises to a single open section.

The Core Loop

  1. Clone last week's session (or create a new one).
  2. Check the pre-class gear list and set out your plates.
  3. During class, log weight and effort for each track as it ends.
  4. Tap Complete Workout when done.
  5. Next week, clone again — your weights are waiting.

Creating a New Session

Choose your workout type — the app builds the right structure for you.

  1. From Home, tap Start New Session.
  2. Fill in session details:
    • Workout name — Edit the default name or enter your own
    • Workout type — BodyPump®, BodyPump® Heavy, Bodybuilding, or Custom
    • Date — Select the workout date
    • Notes — Optional session notes
  3. Tap Create Blank Session to start.

Bodybuilding Options

When selecting Bodybuilding, choose which muscle groups to include:

  • Tap Visible groups to select or deselect body parts
  • Only selected groups appear in your workout
  • You can change this later from session detail

BodyPump® Heavy — Block Designer

A block is a group of tracks done in sequence before a full rest — for example, Chest + Back + Squats cycled through together. The default plan has five blocks (Warmup + four working blocks). To customize:

  1. Tap Block combinations to open the block designer.
  2. Tap any track name within a block to swap it for a different track.
  3. Tap + Add block to add a new block.
  4. Tap − on a block to remove it.
  5. Tap Reset to restore the default plan.
  6. Tap Save when done.

You can also redesign blocks from within an open session using Redesign blocks in session detail.

Logging Your Workout

Tap an exercise row to log weight and effort — everything else is automatic.

Viewing Your Workout

When you open a session, you'll see:

  • Workout name and type at the top
  • Date — tap Edit to change it
  • Pre-class gear checklist (for BodyPump® sessions)
  • Exercises organized by track or body part
  • Session notes at the bottom

Exercise Sections

  • Tap the chevron to expand or collapse a section
  • Tap + to add an exercise to a section
  • Tap the checkmark in a section header to complete all exercises in that section at once

Adding Weight and Effort

  1. Tap an exercise row to open the entry editor.
  2. Enter Weight using the number pad or quick-weight chips, and Effort (1–10) — see below.
  3. Tap the LB / KG chip beside the weight field to switch units.
  4. Add optional Notes for that exercise.
  5. Weight and effort auto-save as you enter them.

Understanding Effort (RPE 1–10)

Effort is a Rate of Perceived Exertion (RPE) score — how hard the exercise felt:

  • 1–4 — Easy, warm-up level; you could keep going indefinitely
  • 5–6 — Moderate, working but comfortable
  • 7–8 — Hard; the last few reps are a challenge
  • 9–10 — Near-maximum or maximum effort

For BodyPump® working tracks, most people land between 6 and 9.

Marking Exercises Complete

Both weight and effort (1–10) are required before you can mark an exercise complete. If either is missing, the completion circle won't respond — enter both values first.

  • Tap the circle icon next to an exercise to toggle completion
  • Tap the checkmark in a section header to complete all exercises in that section at once

Quick Weight Chips

  • Shown in the entry editor for barbell and dumbbell exercises
  • Based on your configured plate and dumbbell options in Settings
  • If chips are missing, check Settings → Gear Setup

Previous Weight Recall

When you tap an exercise, PumpTrack shows your previous weight — "Last used: X lb" — pulled from the most recent matching session.

Previous weights are matched by track name and exercise order. Use Clone workout rather than creating a fresh session to keep matching reliable.

Working with Exercises

Add, change, reorder, and remove exercises within any session.

Adding an Exercise

  1. Tap + in any section.
  2. Search the library for an existing exercise, or tap Custom to enter a new name.
  3. The exercise is automatically added to your library for future sessions.

Changing an Exercise

  1. Tap the exercise row to open the editor.
  2. Tap Change exercise....
  3. Select a replacement from the library. Your weight and effort are preserved unless you clear them.

Reordering Exercises

  • Long-press and drag a row to reorder exercises within a section

Removing an Exercise

  1. Tap the exercise row to open the editor.
  2. Scroll down and tap Remove exercise.
  3. Confirm when prompted.

Pre-Class Gear Checklist

One consolidated list of everything you need before class — no guesswork.

The gear checklist appears at the top of any BodyPump® or BodyPump® Heavy session. It calculates exactly which plates and dumbbells you need for the entire workout — set everything out once before class, not between tracks.

Setup (Do This Once)

In Settings → Gear Setup:

  1. Set Barbell weight input to match how you log weights:
    • Per side — you enter the weight on each side (20 kg = 20 per side)
    • Total barbell — you enter the full loaded weight including the bar
  2. Under Available plates, add every plate size at your gym.
  3. Under Available dumbbells, add the dumbbell weights available to you.

How the Math Works

For each exercise with a weight logged, the checklist calculates the plate combination that makes up that weight. It then consolidates across the whole session. Reuse is accounted for: if two sequential exercises share a plate configuration, those plates are counted once — you carry only what you actually need.

When the Checklist Looks Wrong

  • Nothing appears: Gear inventory isn't configured or no weights are logged yet. Go to Settings → Gear Setup and add plates and dumbbells.
  • Wrong plate breakdown: Check that barbell input mode (per side vs. total) matches how you're entering weights.
  • An exercise is missing: Only exercises with a weight entered show up. Log a weight first.

Completing a Workout

Finish, clone, refresh, or delete a session.

Complete Workout

  1. Scroll to the bottom of the session.
  2. Tap Complete Workout.
  3. All eligible exercises are marked complete and the session is saved to history.

Clone a Workout

  1. Tap More....
  2. Select Clone workout.
  3. A copy is created with all exercises and last session's weights, but completion cleared.

Refresh from Latest

Updates weights in the current session from your most recent matching workout — without creating a new session:

  1. Tap More....
  2. Select Refresh from latest.
  3. Matching exercises are updated with previous weights and effort.

Delete a Workout

  1. Tap More....
  2. Select Delete workout.
  3. The workout moves to "removed" status. You can recall it within 30 days from History.

Viewing History

Browse past sessions, edit historical data, and recover deleted workouts.

Access View Full History from the Home screen.

History Features

  • Filter by workout type — All / BodyPump® / BodyPump® Heavy / Bodybuilding / Custom
  • Show removed — Toggle to see deleted workouts you can recall
  • Export CSV — Export your full history as a spreadsheet
  • Edit mode — Swipe to delete or permanently remove workouts

Editing a Past Workout

History is read-only by default to prevent accidental edits. To modify a past session:

  1. Tap the workout in History.
  2. Tap Edit in the top-right.
  3. Make your changes — they save automatically.

To experiment without affecting the original, use Clone instead of Edit.

Recalling a Deleted Workout

  1. Enable Show removed in History.
  2. Tap the deleted workout.
  3. Tap Recall to restore it.

Body Measurements

Log and track measurements over time alongside your training.

Adding a Measurement

  1. From Home, tap Measurements.
  2. Tap +.
  3. Enter the date, unit (inches or cm), body weight unit, and values for any body parts: weight, waist, hips, chest, shoulders, upper arm, forearm, thigh, calf, neck.
  4. Add optional notes, then tap Save.

Viewing Progress

  • Each entry shows change vs. the previous measurement
  • Entries are sorted by date
  • Tap any entry to view details or edit

Exercise Catalog

Manage your library of movements — rename, browse, and review usage.

The exercise catalog is your library of movements. Every exercise you've added to a session — from suggestions or entered as a custom name — lives here.

Accessing the Catalog

From Home, tap Exercise catalog.

Features

  • View all exercises in your library
  • Tap an exercise to rename it — the name updates across all past and future sessions that reference it
  • Exercise names must be unique (case-insensitive)

Rebuilding Suggested Exercise Lists

The app suggests exercises for each track based on what you've used there before. To rebuild after renaming exercises or changing habits:

  1. Go to Settings → Maintenance.
  2. Tap Rebuild suggested exercise lists from workout history.
  3. Confirm when prompted.

Settings

Configure weight units, gear inventory, and calendar preferences.

Access Settings from the gear icon on the Home screen.

Default Weight Unit

Choose LB or KG as the default for new exercise entries. You can still override the unit per exercise in the entry editor.

Switching the unit does not convert the number. Tapping the unit chip retags the label only — 45 LB becomes 45 KG, not 20.4 KG. Use this to correct a mislabeled entry. To convert, switch the unit first, then update the number manually.

Workout Calendar

  • Show Workout Calendar on Home — Toggle visibility
  • Workout type colors — Customize the color for each workout type

Gear Setup

Configure equipment for the pre-class gear checklist:

  • Barbell weight input — Per side (e.g., 20 kg per side) or Total barbell (full loaded weight)
  • Available plates — Add every plate size at your gym
  • Available dumbbells — Add dumbbell weights; these also power the quick-weight chips in the entry editor

Maintenance

  • Rebuild suggested exercise lists — Refresh exercise suggestions based on your history

Exporting Data

Export your full history as a CSV spreadsheet.

  1. Go to View Full History.
  2. Tap Export CSV in the toolbar.
  3. Choose how to share: AirDrop, email, save to Files, and more.

The CSV includes all sessions, exercises, weights, effort scores, and notes.

Data Management and Safety

How data is stored and how to protect it.

PumpTrack stores all data locally on your device. Until iCloud sync is added, your data is not automatically backed up anywhere else. Export to CSV regularly and save copies to cloud storage.

Before You Delete

  • Deleting a workout moves it to removed status first
  • Removed workouts can be recalled from History within 30 days
  • Permanent deletion cannot be undone

Recommended Backup Routine

  1. Export CSV from View Full History once per week (or after major training blocks).
  2. Save to cloud storage: iCloud Drive, Google Drive, Dropbox, etc.
  3. Use a consistent filename like PumpTrack_YYYY-MM-DD.csv.

Data Quality Tips

  • Keep exercise names consistent to improve suggested lists
  • Avoid duplicate names with different capitalization
  • Run Rebuild suggested exercise lists after many renames

Troubleshooting

Solutions to the most common issues.

I can't mark an exercise complete

Both weight and effort (1–10) are required. Confirm both are entered — effort must be a number from 1 to 10.

Quick-weight chips are missing

Chips appear only for barbell and dumbbell exercises. Check that plates and/or dumbbells are configured in Settings → Gear Setup.

Previous weights aren't showing

Weights are matched by track name and exercise order. If you started a fresh session instead of cloning, changed exercise order, or renamed an exercise, the match breaks. Use Clone workout to preserve matching reliably.

The gear checklist is empty or wrong

See Pre-Class Gear Checklist. Most common causes: gear inventory not configured, wrong barbell input mode, or no weights logged yet.

A deleted workout is not visible

Go to View Full History and enable Show removed. If more than 30 days have passed, the workout may no longer be recallable.

The unit on an entry looks wrong

Tap the exercise row and use the unit chip to retag it. This changes the label only — update the number manually if needed.

Suggested exercises seem outdated

Go to Settings → Maintenance and run Rebuild suggested exercise lists from workout history.

FAQ

Quick answers to the questions that come up most often.

How do I set up for my first BodyPump® class?

Go to Settings → Gear Setup first: set your barbell input mode and add your plate and dumbbell sizes. Then create a new BodyPump® session. After your first class, clone it each week — your exercise order and previous weights carry forward automatically.

Why aren't my previous weights showing?

Weights are matched by track name and exercise position. If you created a fresh session instead of cloning, changed exercise order, or renamed an exercise, the match breaks. Always use Clone workout to keep history matching reliable.

Why can't I mark an exercise complete?

Both weight and effort (1–10) are required. Enter both values and the completion circle becomes tappable.

I switched the unit from LB to KG and the number looks wrong. What happened?

Switching the unit retags the entry — it doesn't convert the number. 45 LB becomes 45 KG, not 20.4 KG. Tap the unit chip again to switch back, or update the number manually to the correct converted value.

The gear checklist looks wrong. What should I check?

First, confirm your plate and dumbbell inventory is set in Settings → Gear Setup. Second, check that your barbell input mode (per side vs. total) matches how you enter weights. Exercises with no weight logged don't appear in the checklist.

I deleted a workout by accident. Can I recover it?

Yes, within 30 days. Go to View Full History, enable Show removed, tap the workout, and tap Recall.

Does PumpTrack auto-save?

Yes. Weight, effort, notes, unit changes, and completion state all save automatically. You can leave mid-session and everything will be there when you return.

Can I use PumpTrack for non-BodyPump® workouts?

Yes. Use Bodybuilding for muscle-group-based lifting or Custom for anything else. Both support multiple exercises per section and the same weight/effort logging.

What's the fastest way to repeat a workout?

Open the previous workout from Home or History, tap More..., and choose Clone workout. The clone opens with all exercises and last week's weights pre-filled.

Can I export my data?

Yes. Use Export CSV from the full History screen. The file includes all sessions, exercises, weights, effort scores, and notes.

Glossary

Definitions for terms used throughout this guide.

Track
A named portion of a BodyPump®-style session targeting one body part — for example, Squats, Chest, or Core. Each track is one collapsible section in the app.
Block
A group of tracks done in sequence without full rest between them, used in BodyPump® Heavy sessions. For example, Chest + Back + Squats cycled together before a rest.
Entry
A single logged instance of an exercise in a session — includes weight, unit, effort score, and optional notes.
Effort (RPE 1–10)
Rate of Perceived Exertion: your subjective intensity rating where 1 is barely any effort and 10 is maximum. Both weight and effort are required to mark an exercise complete.
Clone Workout
Creates a copy of an existing workout with the same exercise structure and previous weights, but with completion cleared. The recommended way to repeat a weekly session.
Refresh from Latest
Updates matching exercises in the current session with your most recent logged values from a similar past workout.
Removed Workout
A soft-deleted workout that can be recalled from History within 30 days before permanent deletion.
Exercise Catalog
Your library of exercise names, used for search, reuse, and consistency across sessions.
Per Side
Barbell input mode where you enter the weight loaded on each side of the bar. The gear checklist uses this to calculate the correct plate combination.

Class-Day Quick Reference

A three-phase checklist for before, during, and after class.

Before Class (5 minutes)

  • Open today's session or clone last week's from Home.
  • Expand the Pre-class gear checklist at the top of the session.
  • Set out your plates and dumbbells based on the checklist.
  • Glance at last week's weights — know what you're aiming for.

During Class (after each track)

  • When a track ends, tap that section to expand it.
  • Enter weight and effort (1–10) while it's fresh — don't wait until the end of class.
  • Tap the section checkmark to complete the whole track at once.

After Class (2 minutes)

  • Add session notes — what felt strong, what to adjust, weights to change next week.
  • Tap Complete Workout to lock in the session.
  • Verify it appears in View Full History.

Tips and Best Practices

Habits that make PumpTrack work better over time.

For BodyPump® Classes

  • Clone, don't create. Use Clone workout each week instead of starting fresh — exercise order and previous weights carry forward, and previous-weight recall stays reliable.
  • Log each track when it ends, not at the end of class. Memory degrades under fatigue; logging right after each track is far more accurate.
  • Use the section checkmark (tap the track header) to complete all exercises in a track at once.
  • Set up your plate inventory once in Settings so the gear checklist works automatically every week.

For BodyPump® Heavy

  • Design your blocks before class, not during warm-up — open the block designer when creating the session.
  • The gear checklist is especially useful for Heavy — you'll often need more plate combinations than a standard class.
  • You can redesign blocks from within an open session if the structure changes on the day.

For Bodybuilding

  • Select only the muscle groups you're training that day to keep the session view clean.
  • Add multiple exercises per body part using + in each section.
  • Use session notes for supersets, rest times, or anything that doesn't fit in weight/effort fields.

General Tips

  • The app auto-saves as you type — no manual save needed.
  • Export CSV regularly and save to cloud storage. Until iCloud sync is added, this is your only backup.
  • Effort scores over time tell you whether a weight is getting easier — a clear sign to increase next week.
  • Use the calendar on Home to see workout frequency at a glance.
